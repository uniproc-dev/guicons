# guicons-idea-plugin

RustRover/IntelliJ support for guicons' `icons.gui.toml` manifests: full
LSP integration (completion, hover, goto-definition, find usages, rename,
diagnostics) for the manifest and for icon macros in your Rust code, plus
a native icon browser (floating popup or pinned sidebar tool window) that
searches, previews, and inserts a reference at the caret from either your
manifest's own icons or iconify.design's.
